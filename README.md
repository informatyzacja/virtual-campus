## Getting started

### Requirements

#### Development

- Volta

### Installation

```bash
git clone https://github.com/informatyzacja/virtual-campus.git
cd virtual-campus
pnpm install --frozen-lockfile
```

## Usage

Run in development mode:

```bash
pnpm dev
```

or start the app:

```bash
pnpm start
```
